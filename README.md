# 📝 Task 04 – To-Do Web Application

## 📌 Overview

This project was developed as **Task 04** for the **SkillCraft Technology Web Development Internship**.

The objective of this project is to create a **responsive and user-friendly To-Do Web Application** that helps users efficiently manage their daily tasks. The application allows users to add, edit, organize, complete, and delete tasks while also setting a date and time for each task.

---

# ✨ Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* 📅 Set task date
* ⏰ Set task time
* 📋 Organize tasks in a structured list
* 📱 Fully responsive design
* 🎨 Clean and intuitive user interface

---

# 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

# 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/to-do-web-app.git
```

2. Navigate to the project folder.

3. Open `index.html` in your preferred web browser.

No additional installation or dependencies are required.

---

# 📖 How to Use

1. Enter a new task.
2. Select the task date and time.
3. Click **Add Task** to save it.
4. Edit tasks whenever needed.
5. Mark tasks as completed after finishing them.
6. Delete tasks you no longer need.
7. Organize your daily activities efficiently.
